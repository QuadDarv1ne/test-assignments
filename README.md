## Установка

1. Клонируйте репозиторий:
    ```bash
    git clone https://github.com/ваш-пользователь/test-assignments.git
    ```

2. Перейдите в директорию проекта:
    ```bash
    cd test-assignments
    ```

3. Установите необходимые зависимости:
    ```bash
    pip install -r requirements.txt
    ```
